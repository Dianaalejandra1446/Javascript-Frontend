    Metodo Push

    El método push() es usado para agregar un elemento al final de un arreglo.Digamos que tienes un arreglo de elementos, cada elemento es una cadena que representa una tarea que necesitas realizar. Tendría sentido agregar nuevos elementos al final del arreglo para poder terminar primero las tareas anteriores.

    Digamos que quisiéramos agregar dos o tres elementos a la vez a nuestra lista, ¿Qué haríamos entonces? Resulta que push() puede aceptar que se agreguen varios elementos a la vez.

    Metodo pop

    El método pop en JavaScript es una herramienta útil cuando necesitas eliminar el último elemento de un array. Este método no solo elimina el último elemento, sino que también lo devuelve.

    Sintaxis

    // Declaramos un array vacío
                
    Sintaxis Metodo Push
    
                var frutas = [];
                
                // Utilizamos el método push para agregar elementos al array
                frutas.push("manzana");
                frutas.push("banana");
                frutas.push("uva");
                // Imprimimos el contenido del array
                console.log(frutas); // Output: ["manzana", "banana", "uva"]

                
    
        
                    En este ejemplo, declaramos un array vacío llamado "frutas". Luego, utilizamos el método push para agregar elementos al array. Cada vez que utilizamos push, le pasamos como argumento el elemento que queremos agregar al final del array. En este caso, agregamos las frutas "manzana", "banana" y "uva". Al imprimir el contenido del array, obtenemos el resultado ["manzana", "banana", "uva"].
                
    
            
                    
    Sintaxis pop
    
                // Creamos un array de nombres
                let nombres = ['Ana', 'Juan', 'María', 'Pedro'];

                // Utilizamos el método pop para eliminar el último elemento del array
                let ultimoNombre = nombres.pop();

                // Imprimimos el último elemento eliminado y el array actualizado
                console.log('Último nombre eliminado:', ultimoNombre);
                console.log('Array actualizado:', nombres);
                
    
                    En este ejemplo, utilizamos el método pop() para eliminar el último elemento del array nombres. El método pop() devuelve el elemento eliminado, que almacenamos en la variable ultimoNombre. Luego, imprimimos el elemento eliminado y el array actualizado usando console.log().
